== Description ==

Speed up you site by creating additional database indexes. No configuration needed, just install and activate. To remove additional indexes, just deactivate plugin. Page Speed Optimizer is absolutely free.

== Installation ==

1. Unzip `page-speed-optimizer` folder to the `/wp-content/plugins/` directory
. Activate the plugin through the 'Plugins' menu in WordPress
